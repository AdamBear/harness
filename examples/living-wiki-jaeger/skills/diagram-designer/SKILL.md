# Diagram Designer

Create diagrams that explain relationships, decisions, risks, or workflow
phases. Prefer readable Mermaid source with stable labels and simple node ids.

If draw.io MCP tooling is available, describe the same diagram as draw.io-ready
structured intent. If it is unavailable, keep the Mermaid artifact complete and
useful on its own.

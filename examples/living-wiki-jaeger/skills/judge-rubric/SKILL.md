# Judge Rubric

Score workflow outputs with criteria that match the task. Use approved only
when evidence is sufficient and risks are addressed. Use needs_human_review,
revise, or rejected when the result requires human judgement or more work.

Return machine-readable rubric fields with criterion ids, labels, scores, and
rationales.
